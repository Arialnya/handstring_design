简易智能手环app开发文档

## 1. 项目概述
手机与手环通过蓝牙连接，以类似json方式传输数据给手机端
app页面提供可视化图表，展示最近接收的数据，包括心率、血氧、环境亮度以及吃药记录

## 2. 功能模块
### 2.1 数据采集模块
- 通过蓝牙接收手环数据
- 支持的数据类型：
  - 心率数据
  - 血氧数据
  - 环境亮度数据
  - 吃药记录数据
  - 睡眠数据


### 2.2 数据存储模块
- 本地数据库存储历史数据
- 支持数据导出功能
- 数据备份功能

### 2.3 数据展示模块
- 实时数据展示
- 历史数据统计
- 数据可视化图表
- 吃药提醒和记录

## 3. 页面设计
### 3.1 主页面
- 顶部状态栏：显示蓝牙连接状态
- 数据概览卡片：显示当前心率、血氧、环境亮度等关键指标
- 快捷功能入口：数据统计、吃药记录、设置等
- 设计风格：简洁现代，以蓝色为主色调

### 3.2 数据统计页面
- 时间选择器：支持按日/周/月查看数据
- 数据图表展示：
  - 心率趋势图
  - 血氧趋势图
  - 环境亮度变化图
  - 睡眠质量分析图
  - 吃药记录统计图
- 数据详情列表

### 3.3 吃药记录页面
- 药品信息管理
- 吃药提醒设置
- 吃药记录查看
- 用药统计图表

### 3.4 设置页面
- 蓝牙连接设置
- 数据同步设置
- 通知设置
- 个人信息设置

## 4. 技术架构
### 4.1 前端技术
- 开发框架：React Native
- UI组件库：React Native Paper
- 图表库：Victory Native

### 4.2 数据存储
- 本地数据库：SQLite
- 数据缓存：AsyncStorage

### 4.3 蓝牙通信
- 蓝牙库：react-native-ble-plx
- 数据格式：JSON
